# Flora-Cycle
The flowers that are offered in temples are usually discarded after one use, but some companies recycle these flowers. So, this is a website that connects temples and flower recycling companies. Someone's trash is Someone's Gold.
> **Note:** This repository contains the initial version of FloraCycle, built with plain HTML and JavaScript. Due to merge conflicts arising from an unstructured codebase, the project has been migrated to a new repository using React for better collaboration and code maintainability.
>
> All further development is happening here: https://github.com/Gungun210/Flora_cycle
